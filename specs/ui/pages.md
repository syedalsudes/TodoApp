# UI Pages Specification

## Pages

### Authentication Pages
- /login
- /signup

Features:
- Email-based login/signup using Better Auth
- Redirect to /tasks after login

---

### Tasks Page
Route: /tasks

Features:
- Display list of tasks
- Add new task form
- Edit task
- Delete task
- Toggle completed status

## UI Rules
- Responsive design
- Clean and minimal layout
- Loading and error states must be handled

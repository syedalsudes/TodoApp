# AI Agent Overview

The Todo Assistant is an AI agent responsible for managing tasks
using natural language instructions.

The agent does not directly access the database.
Instead, it invokes predefined tools that execute verified backend logic.

## Responsibilities
- Understand user intent
- Select the correct tool
- Pass validated parameters
- Return a human-readable response

## Supported Actions
- Add task
- List tasks
- Update task
- Complete task
- Delete task
